# Phyllux Framework

**Ethical IP & 4-Tier Licensing for Independent Inventors**

## Overview

IP protection and licensing system for independent inventors: prior-art templates, 4-tier licensing (Pioneer / Mission-Aligned / Commercial / Free), and governance for shared innovation.

**Who it's for:** Inventors, researchers, companies wanting fair terms.

## 4-Tier Licensing

| Tier | Who | Terms |
|------|-----|-------|
| **Pioneer** | Civilization-scale missions (space, AGI, longevity) | Minimal upfront + 0% royalty |
| **Mission-Aligned** | Medical, climate, education, humanitarian | $10K–100K + 1–3% royalty |
| **Commercial** | For-profit use | $100K–1M + 3–5% royalty |
| **Free** | Research, education, non-commercial | $0 (citation required) |

**Full model:** [TEMPLATES/TEMPLATE_License_4Tier.md](TEMPLATES/TEMPLATE_License_4Tier.md)
